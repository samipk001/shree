# Shree Trading Website

This repository contains the official, premium, responsive website for **Shree Trading**, a hardware and machinery showroom based in Birtamode, Jhapa, Nepal.

## About Shree Trading
Shree Trading has been standing for almost 20 years (since 2006) as Eastern Nepal's premier choice for:
- Professional Power Tools (Authorized Dealer for INGCO, Bosch, Makita, DeWalt)
- Heavy Industrial Machinery & Equipment (Air Compressors, Diesel Generators, Motors - Authorized Dealer for Libang)
- High-quality Hand Tools & Hardware
- Agricultural Pumps & Spraying Systems
- Showroom Product Demonstrations & Expert Consultations

## Tech Stack
- HTML5 / Semantic Layouts
- Bootstrap 5 (Styling & Responsive Grid)
- Custom Vanilla CSS
- Owl Carousel
- AOS (Animate on Scroll)
- Iconify Icons

## Structure
- `src/html/index.html` - Main store landing page
- `src/html/contact.html` - Quote request and contact portal
- `src/html/404.html` - Custom error page
- `src/html/privacy-policy.html` - Customer privacy statements
- `src/html/terms-and-conditions.html` - Sales, manufacturer warranty, and store policies
- `src/assets/images/logos/` - Custom vector SVG brand logos
- `src/assets/images/brands/` - Partner and dealer brand SVGs (INGCO, Libang, Bosch, Makita, DeWalt)
- `src/assets/images/work/` - High-quality generated photography showcasing actual machinery and power tools
